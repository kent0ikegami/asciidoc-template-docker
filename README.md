
# asciidoc-template

Asciidoc を使った仕様書の構成管理・CI/CD環境の雛形です。

詳細は[記事](https://qiita.com/ynitto/items/fa44aae90c927e52613c)を参照してください。


## 導入

git
vscode
npm

vscode拡張
```
{
  "recommendations": [
    "asciidoctor.asciidoctor-vscode",
    "taichi.vscode-textlint",
    "ms-vscode.live-server",
    "hediet.vscode-drawio"
  ]
}
```

```
npm run install
```
